We are a group of BSIT students from FAITH Colleges who collaborated on this project as part of our Calculus-Based Physics course. This website is the result of our combined efforts to apply programming skills to solve real-world physics problems. From designing the user interface to implementing the core calculator functions, each of us contributed to building a tool that’s not only functional but also educational. Get to know the team behind the code — future developers passionate about learning, technology, and making physics easier to understand.
